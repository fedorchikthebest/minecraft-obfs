# minecraft-obfs
Проект создан для маскировки трафика под трафик майнкрафта.

# Принцип работы
На этапе обмена ключами прокси проверяет наличие в общем секрете блока, зашифрованного ключём, принадлижащем прокси.
Если он этот блок находит, то весь трафик клиента будет перенаправляться на указанный в коде прокси. Если такого блока нет,
то весь трафик перенаправляется на выбранный майнкрафт сервер.

# Требования к майнкрафт серверу
* Включенный online-mode
